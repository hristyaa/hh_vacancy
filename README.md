# Приложение для получения информации о вакансиях с платформы hh.ru в России.

## Описание:

Программа получает информацию о вакансиях с платформы hh.ru в России, сохраняет ее в файл и позволяет удобно работать с ней.

## Установка:

Клонируйте репозиторий:
```
git clone https://github.com/hristyaa/hh_vacancy.git
```

## Запуск:

```
python main.py
```

## Функциональные возможности:
Функциональный код разбит на модули: модуль для взаимодействия с API, модуль для взаимодействия с файлами, модуль для взаимодействия с вакансиями.
1. **Модуль для взаимодействия с API - hh_api.py**.
Создан абстрактный класс VacancyAPI для работы с API сервиса с вакансиями. 
Реализован класс HeadHunterAPI, наследующийся от абстрактного класса, для работы с платформой hh.ru. 
Класс подключается к API и получает вакансии.


2. **Модуль для взаимодействия с вакансиями - vacancy.py**
Создан класс для работы с вакансиями Vacancy. 
В этом классе определены атрибуты, такие как название вакансии, ссылка на вакансию, зарплата, описание. 
Класс поддерживает методы сравнения вакансий между собой по зарплате и 
валидирует данные, которыми инициализируются его атрибуты.


3. **Модуль для взаимодействия с файлами - file_saver.py**
Определен абстрактный класс FileSaver, который обязывает реализовать методы для добавления вакансий в файл, получения данных из файла по указанным критериям и удаления информации о вакансиях. 
Создан класс JSONSaver для сохранения информации о вакансиях в JSON-файл.

В программе реализована функция для взаимодействия с пользователем. 
Функция взаимодействует с пользователем через консоль. 
Возможности этой функции следующие:
-ввод поисковый запрос для запроса вакансий из hh.ru;
-ввод количество вакансий для вывода в топ N;
-ввод ключевых слов для вывода в топ N;
-получение топ N вакансий по зарплате;
-получение вакансии с ключевым словом в описании;
-запись всех полученных по запросу ваканский в JSON-файл и возможность его очищения по желанию.
